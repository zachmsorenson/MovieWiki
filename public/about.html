TV/Movie Wiki

This is a dynamic web application that retrieves information about television shows and movies from an SQL database. Users may search for tv shows, movies, actors, directors, etc. and pull up a page about it accordingly.

Express

Express is a Node.js framework that allows developers to set up middleware to respond to HTTP Requests and to dynamically render HTML pages by passing arguments to templates.

jQuery

jQuery is a JavaScript framework. It is a Document Object Model (DOM) manipulation library that simplifies the syntax for finding, selecting, and manipulating DOM elements.

Poster API

We used this API to retrieve the television show and movie posters and to retrieve images of people.

Bootstrap

Bootstrap is a free and open-source library for designing websites and web applications. We used a Bootstrap template to design our "About the Project" page.

SQLite

SQLite is a relational database management system. We queried an sqlite3 database to retrieve information on tv shows/movies and people.
